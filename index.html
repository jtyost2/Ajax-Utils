Demo of Repleace Location jQuery Plugin

Replace Location with a Link

This pagraph will go away when you click the link.

Replace the above pargraph with data.

Code

<p id="replaceThisPargraphButton">
This pagraph will go away when you click the link.
</p>

<a
	href="./sample_ajax.php"
	class="replace_location middle"
	replaceLocation="#replaceThisPargraphButton">
	Replace the above pargraph with data.
</a>

Replace Location with a Select Element

This pagraph will go away when you change the select.

Code

<p id="replaceThisPargraphSelect">
This pagraph will go away when you change the select.
</p>

<form>
	<select
		name="select"
		class="replace_location"
		href="./sample_ajax.php"
		replaceLocation="#replaceThisPargraphSelect">
		
		<option value="first">First</option>
		<option value="second">Second</option>
		<option value="third">Third</option>
		<option value="fourth">Fourth</option>
	</select>
</form>
