Demo of Replace Location jQuery Plugin

Replace Location with a Link

This paragraph will go away when you click the link.

Replace the above paragraph with data.

Code

<script type="text/javascript">
	$(document).ready(function (){
		$('.replace_location').replace_location({ajaxType: 'GET'});
	});
</script>
<p id="replaceThisParagraphButton">
This pagraph will go away when you click the link.
</p>

<a
	href="./sample_ajax.php"
	class="replace_location middle"
	data-replace-location="#replaceThisParagraphButton">
	Replace the above paragraph with data.
</a>
